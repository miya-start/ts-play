# ts-play

A better "fiddle" for TypeScript. Work in progress.

## Building and running

### Front-end

- `npm install`
- `./setup-monaco.sh`
- `webpack`

### Server

- `npm install`
- `npm run build`

### Running (for development)

- in a container: `docker-compose up` 
- locally: run  `node server` in the `dist` directory
- go to `http://localhost:2080/`
